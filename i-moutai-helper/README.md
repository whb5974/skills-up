## 🚀 快速开始

### 环境要求

- Python 3.10+
- 飞书账号 (用于接收通知)
- 高德地图 API Key (可选，用于门店查询)

### 安装步骤

```bash
# 1. 克隆项目
cd /home/ghost/.openclaw/workspace
git clone <repo> i-moutai-helper
cd i-moutai-helper

# 2. 创建虚拟环境
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# 或 venv\Scripts\activate  # Windows

# 3. 安装依赖
pip install -r requirements.txt

# 4. 配置参数
cp config.yaml.example config.yaml
# 编辑 config.yaml 填入你的配置

# 5. 运行程序
python main.py
```

### 后台运行

```bash
# 使用 nohup
nohup python main.py > output.log 2>&1 &

# 查看日志
tail -f output.log

# 停止服务
pkill -f "python main.py"
```

## ⚠️ 常见问题

### Q: 会被封号吗？

A: 不会。本工具仅提供提醒和统计功能，所有申购操作由用户手动完成，完全合规。

### Q: 能提高中签率吗？

A: 工具本身不能提高中签率，但能确保你不错过申购时间，间接提高成功率。

### Q: 需要一直运行吗？

A: 是的。提醒服务需要后台运行，建议使用 nohup 或 systemd 部署。

### Q: 支持多账号吗？

A: v1.0 支持单账号，v1.1 将支持多账号。
