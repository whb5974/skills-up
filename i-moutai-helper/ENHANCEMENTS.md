## 📋 新增功能

### 1️⃣ 多账号管理 👨‍👩‍👧‍👦

**功能说明：** 支持添加家人账号，一起申购提高中签率

**使用命令：**

```bash
# 查看账号列表
python3 main.py account list

# 添加账号
python3 main.py account add --phone "13900139000" --name "李四" --relation "配偶"

# 移除账号
python3 main.py account remove --phone "13900139000"

# 生成申购计划
python3 main.py account plan --product "飞天 500ml" --stores "朝阳店，海淀店，东城店"
```

**测试结果：**

```

   2. 李四 (13900139000)
   3. 王五 (13800138001)
==================================================

📊 家庭汇总:
   账号数：3
   总申购：7 次
   总中签：1 次
   家庭中签率：14.3%
```

**使用示例：**

### 4️⃣ 申购冲突检测 ⚠️

**功能说明：** 检测多账号是否选择同一门店，避免内部竞争

**检测规则：**

- 同一门店多个账号申购 → 发出警告
- 建议分散到不同门店
- 自动轮换分配门店

## 🎯 中签率提升策略

### 多账号策略

| 策略 | 说明 | 效果 |
|------|------|------|
| 家人账号 | 配偶/父母/子女一起申购 | 3 倍机会 |
| 分散门店 | 各账号选择不同门店 | 避免竞争 |
| 坚持申购 | 每日不间断 | 提高活跃度 |
| 冷门产品 | 选择竞争小的产品 | 中签率高 |

### 门店选择建议

```
推荐策略：
1. 选择 3-5 家门店
2. 避开核心商圈 (竞争大)
3. 选择交通便利的门店
4. 各账号固定门店 (便于统计)
```
