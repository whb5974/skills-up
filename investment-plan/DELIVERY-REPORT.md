## 🎯 核心功能演示

### 1. 投资日报

```bash
python3 analyze.py --quick
```

输出：
- 总资产、总收益
- 各持仓盈亏
- 风险状态
- 操作建议

### 2. 估值分析

```bash
python3 analyze.py --valuation
```

输出：
- PE/PB/PEG 估值
- 低估/合理/高估判断
- 操作建议（加仓/持有/减仓）

### 3. 智能调仓

```bash
python3 analyze.py --rebalance
```

输出：
- 目标配置 vs 当前配置
- 偏离度分析
- 调仓金额计算
- 具体操作建议

### 4. 股票筛选

```bash
python3 analyze.py --screen 价值股
```

支持策略：
- 价值股（低 PE、高 ROE）
- 成长股（高增长）
- 蓝筹股（大盘稳健）
- 高股息（分红稳定）
- 低估值（历史百分位低）

### 5. 可视化图表

```bash
python3 analyze.py --charts
```

生成：
- 资产配置图
- 盈亏分布图
- 风险等级图

### 6. 完整分析

```bash
python3 analyze.py
```

一次性运行所有分析！
