## 🛠️ 核心功能详解

### 1. 📊 投资组合跟踪器 (`portfolio-tracker.py`)

**功能：**
- 持仓管理（买入/卖出）
- 成本核算
- 盈亏计算
- 交易记录

**使用方法：**
```bash
# 初始化组合
python3 portfolio-tracker.py --init

# 添加持仓
python3 portfolio-tracker.py --add

# 卖出持仓
python3 portfolio-tracker.py --sell

# 更新价格
python3 portfolio-tracker.py --update

# 查看报告
python3 portfolio-tracker.py --report
```

**输出示例：**
```

600519   贵州茅台          20     1800.00    1800.00    +0(0.0%)   36.0%
```

### 3. 💰 估值分析 (`valuation.py`)

**功能：**
- PE（市盈率）估算
- PB（市净率）估算
- PEG（市盈率相对盈利增长比率）
- 估值状态判断（低估/合理/高估）
- 操作建议

**使用方法：**
```bash
python3 analyze.py --valuation
# 或
python3 src/valuation.py
```

**输出示例：**
```
📊 持仓估值分析报告
============================================================
📈 估值分布:
   🟢 低估：0 只
   🟡 合理：6 只
   🔴 高估：0 只

📋 详细分析:
----------------------------------------------------------------------
代码       名称           价格       PE     PB    PEG       估值       建议
----------------------------------------------------------------------
600519   贵州茅台       1800.00     28.0    4.2   1.87 🟡 合理   继续持有
600036   招商银行         35.00      6.0    0.9   0.40 🟡 合理   继续持有
----------------------------------------------------------------------
```

**估值标准：**
- **低估**（🟢）：PE < 行业平均 × 0.7，建议加仓
- **合理**（🟡）：PE 在合理区间，继续持有
- **高估**（🔴）：PE > 行业平均 × 1.3，考虑减仓

### 4. 🔄 智能调仓 (`rebalancer.py`)

**功能：**
- 检测配置偏离
- 生成调仓建议
- 计算调仓金额
- 预估交易成本

**使用方法：**
```bash
python3 analyze.py --rebalance
# 或
python3 src/rebalancer.py
```

**输出示例：**
```
🔄 智能调仓建议
============================================================
💰 总资产：100,000.00 元
⚙️ 调仓阈值：5%

📊 配置对比:
----------------------------------------------------------------------
类别           目标        当前        偏离          操作          金额
----------------------------------------------------------------------
股票          40.0%      39.5%      -0.5% ✅ 买入 500 元              +500
基金          25.0%      24.9%      -0.1% ✅ 保持                    +50
债券          15.0%      15.0%      +0.0% ✅ 保持                     +0
现金          15.0%      15.6%      +0.6% ✅ 卖出 550 元              -550
另类           5.0%       5.0%      +0.0% ✅ 保持                     +0
----------------------------------------------------------------------

✅ 当前配置合理，无需调仓
```

**调仓规则：**
- 偏离 > 5%：建议调仓
- 偏离 < 5%：保持观察
- 优先使用新增资金调仓
- 减少卖出操作，降低税费

### 5. 🔍 股票筛选 (`screener.py`)

**功能：**
- 预设筛选条件（价值股/成长股/蓝筹股/高股息/低估值）
- 自定义筛选条件
- 基本面分析
- 排序输出

**使用方法：**
```bash
# 列出所有预设
python3 src/screener.py --list

# 筛选价值股
python3 analyze.py --screen 价值股

# 筛选成长股
python3 analyze.py --screen 成长股

# 筛选高股息
python3 analyze.py --screen 高股息
```

**预设条件：**

| 预设 | 条件 |
|------|------|
| **价值股** | PE≤15, PB≤2, 股息率≥3%, ROE≥10% |
| **成长股** | 营收增长≥20%, 利润增长≥25%, PE≤40 |
| **蓝筹股** | 市值≥100 亿，ROE≥15%, 负债率≤60%, PE≤25 |
| **高股息** | 股息率≥5%, 分红比例≤70%, 连续≥3 年 |
| **低估值** | PE 历史百分位≤30%, PB 历史百分位≤30% |

**输出示例：**

## ⚙️ 配置说明

### 调仓阈值配置
编辑 `src/rebalancer.py`:
```python
REBALANCE_THRESHOLD = 0.05  # 5% 偏离触发调仓
```

### 目标配置比例
编辑 `src/rebalancer.py`:
```python
TARGET_ALLOCATION = {
    '股票': 0.40,
    '基金': 0.25,
    '债券': 0.15,
    '现金': 0.15,
    '另类': 0.05,
}
```

### 筛选条件配置
编辑 `src/screener.py`:
```python
SCREEN_PRESETS = {
    '价值股': {
        'pe_max': 15,
        'pb_max': 2,
        'dividend_yield_min': 0.03,
        'roe_min': 0.10,
    },
    # ... 更多预设
}
```

## 🚀 进阶使用

### 1. 自定义股票池
编辑 `src/screener.py` 的 `get_default_stock_pool()` 方法，添加你关注的股票。

### 2. 接入实时数据
修改 `src/valuation.py`，接入 Tushare 或聚宽 API 获取真实财务数据。

### 3. 添加新筛选条件
在 `src/screener.py` 的 `SCREEN_PRESETS` 中添加新的预设条件。

### 4. 自动化运行
配置 cron 定时任务：
```bash
# 每日 9:00 生成日报
0 9 * * * cd /path/to/investment-plan && python3 auto-check.py >> logs/daily.log 2>&1

# 每周五 17:00 完整分析
0 17 * * 5 cd /path/to/investment-plan && python3 analyze.py >> logs/weekly.log 2>&1
```

## ⚠️ 注意事项

1. **数据准确性**
   - 当前使用示例数据/估算数据
   - 实际投资请查阅最新财报和行情
   - 建议接入真实数据源

2. **估值方法**
   - PE/PB 为简化估算
   - 实际估值需考虑行业特性
   - 结合基本面综合判断

3. **调仓成本**
   - 考虑交易手续费
   - 考虑税费影响
   - 避免频繁调仓

4. **投资风险**
   - 工具仅供参考
   - 不构成投资建议
   - 决策需自主判断
