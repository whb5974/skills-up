# Skill: code-review

## 描述
对指定文件进行代码审查，检查潜在问题。

## 审查要点
1. **语法检查**: 确保代码语法正确
2. **错误处理**: 检查是否有适当的错误处理
3. **安全检查**: 查找潜在的安全问题（硬编码密码、注入风险等）
4. **性能问题**: 识别明显的性能问题
5. **代码规范**: 检查是否符合项目规范
6. **边界条件**: 检查边界情况和空值处理

## 输出格式
```
## 审查结果：[文件名]

### ✅ 优点
- ...

### ⚠️ 需要注意
- ...

### ❌ 问题
- [严重] ...
- [一般] ...

### 建议修改
[具体修改建议]
```

## 使用方式
在 Claude Code 中输入：`/skill code-review <文件路径>`
