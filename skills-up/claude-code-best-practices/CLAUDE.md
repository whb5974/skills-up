# CLAUDE.md - 项目持久指令

## 项目上下文
这是 OpenClaw 工作空间，用于运行 AI 助手和自动化任务。

## 编码规范
- **语言**: 优先使用 JavaScript/TypeScript 和 Python
- **风格**: 简洁、高效、可读性强
- **注释**: 关键逻辑必须注释，使用中文注释
- **错误处理**: 所有外部调用必须有错误处理

## 工作流程

### 文件操作
1. 修改前先读取文件理解上下文
2. 使用精确的 edit 操作而非重写整个文件
3. 修改后验证语法正确性

### 代码审查
- 检查边界条件和错误处理
- 确保没有硬编码敏感信息
- 验证 imports/dependencies 正确

### 测试要求
- 新功能必须包含测试用例
- 优先使用现有测试框架
- 确保测试可重复运行

## 安全规则
- ❌ 不要提交 API keys、密码等敏感信息
- ❌ 不要执行破坏性命令（rm -rf 等）
- ✅ 使用 trash 而非 rm 删除文件
- ✅ 外部操作前必须确认

## 沟通风格
- 简洁高效，拒绝废话
- 技术方案优先给出最简可行方案
- 遇到问题先尝试解决，再报告

## 记忆重点
- 记录项目架构决策
- 记录配置变更原因
- 记录遇到的坑和解决方案
