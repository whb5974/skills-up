# 《卜算子·咏梅》对比分析思维导图

## 两首词原文

### 毛泽东《卜算子·咏梅》（1961 年）
```
风雨送春归，飞雪迎春到。
已是悬崖百丈冰，犹有花枝俏。

俏也不争春，只把春来报。
待到山花烂漫时，她在丛中笑。
```

### 陆游《卜算子·咏梅》（南宋）
```
驿外断桥边，寂寞开无主。
已是黄昏独自愁，更著风和雨。

无意苦争春，一任群芳妒。
零落成泥碾作尘，只有香如故。
```

---

## 思维导图：两首《咏梅》异同对比

### 🎯 相同点

#### 1. 词牌相同
- 都使用《卜算子》词牌
- 双调四十四字，上下片各四句

#### 2. 咏物主题
- 都以梅花为歌咏对象
- 借梅抒怀，托物言志

#### 3. 梅花品格
- 都赞美梅花傲雪凌霜的品质
- 都写梅花在恶劣环境中绽放

#### 4. 不争春
- 毛词："俏也不争春"
- 陆词："无意苦争春"
- 都表达梅花不与百花争艳的高洁

---

### 🔄 不同点

#### 1. 创作背景
| 毛泽东 | 陆游 |
|--------|------|
| 1961 年冬，中国困难时期 | 南宋时期，抗金失败后 |
| 革命乐观主义精神 | 个人失意、报国无门 |
| 鼓励人民战胜困难 | 抒发个人怀才不遇 |

#### 2. 梅花意象
| 毛泽东 | 陆游 |
|--------|------|
| 积极乐观 | 孤寂悲凉 |
| "花枝俏"——自信美丽 | "寂寞开无主"——孤独无助 |
| "她在丛中笑"——欣慰豁达 | "独自愁"——哀怨忧愁 |
| 革命者形象 | 失意文人形象 |

#### 3. 情感基调
| 毛泽东 | 陆游 |
|--------|------|
| 昂扬向上 | 低沉哀婉 |
| 豪迈乐观 | 悲愤无奈 |
| 集体主义精神 | 个人主义情怀 |
| 展望未来 | 坚守节操 |

#### 4. 艺术手法
| 毛泽东 | 陆游 |
|--------|------|
| 拟人化："笑" | 拟人化："愁"、"妒" |
| 对比：冰雪 vs 花枝 | 对比：群芳 vs 梅花 |
| 象征：革命者 | 象征：高洁士人 |
| 语言通俗明快 | 语言含蓄深沉 |

#### 5. 结尾意境
| 毛泽东 | 陆游 |
|--------|------|
| "她在丛中笑" | "只有香如故" |
| 与民同乐 | 坚守本心 |
| 融入集体 | 孤芳自赏 |
| 积极向上 | 悲壮坚守 |

## 核心对比总结

```
                    毛泽东                    陆游
                    ↓                        ↓
时代背景 →    新中国困难时期          南宋抗金失败
情感色彩 →    乐观豪迈                悲凉孤寂
梅花形象 →    革命战士                失意文人
精神内核 →    集体主义                个人节操
艺术风格 →    明快昂扬                含蓄深沉
历史意义 →    鼓舞斗志                抒发情怀
```

---

## 学习要点

1. **知人论世**：理解诗词要结合作者生平和时代背景
2. **意象分析**：同一意象（梅花）可承载不同情感
3. **对比阅读**：通过对比深化对作品的理解
4. **托物言志**：中国古典诗词的重要表现手法
