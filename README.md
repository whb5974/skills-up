# Data Analysis Skill

📊 数据分析与可视化工具

## 功能特性

- ✅ 加载和解析数据文件 (CSV, Excel, JSON, SQL)
- ✅ 使用 pandas/numpy 进行数据分析
- ✅ 生成统计摘要和洞察
- ✅ 创建可视化图表 (matplotlib, seaborn, plotly)

## 依赖安装

```bash
pip install pandas numpy matplotlib seaborn plotly
```
