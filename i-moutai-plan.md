## 📊 i 茅台抢购机制分析

### 申购流程

```
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  实名认证    │ →  │  预约申购    │ →  │  等待抽签    │
│ (人脸识别)   │    │ (9:00-21:00)│    │ (21:00 后)   │
└─────────────┘    └─────────────┘    └─────────────┘
                                              ↓
┌─────────────┐    ┌─────────────┐    ┌─────────────┐
│  线下提货    │ ←  │  支付货款    │ ←  │  中签通知    │
│ (7 天内)     │    │ (中签后 24h) │    │ (短信/APP)  │
└─────────────┘    └─────────────┘    └─────────────┘
```

### 关键时间点

| 时间 | 操作 | 说明 |
|------|------|------|
| 09:00 | 申购开始 | 可以提交预约申购 |
| 21:00 | 申购截止 | 停止接受当日申购 |
| 21:00-22:00 | 系统抽签 | 自动摇号 |
| 22:00 后 | 公布结果 | APP/短信通知 |

### 申购规则

1. **实名认证** - 必须完成实名认证和人脸识别
2. **地理位置** - 必须在门店服务范围内
3. **申购限制** - 同一用户同一产品每天只能申购 1 次
4. **门店选择** - 需要提前选择提货门店
5. **中签限制** - 30 天内中签不超过 2 次
